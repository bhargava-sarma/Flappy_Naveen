# Flappy Naveen

A custom Flappy Bird clone built with HTML5 Canvas and JavaScript.

## how to Play
1. Enter your name (it will be remembered).
2. Click "Start Game".
3. Tap, Click, or press Space/ArrowUp to fly.
4. Avoid the pipes!
5. If you crash, you'll see your score and the "You let Naveen down" screen.

## Deployment to Vercel
1. Upload this folder to a GitHub repository.
2. Go to [Vercel](https://vercel.com) and "Add New Project".
3. Import your repository.
4. Click "Deploy". Vercel automatically detects it as a static site.
